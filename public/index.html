機能紹介

集中を、もっとおいしく。

⏱
ポモドーロタイマー

/pp ひとつで開始。総時間を5:1で自動分割し、集中→休憩を管理。

🍅
ステータス自動変更

タイマー中は🍅に。終了後は自動で元に戻します。文言カスタムも可。

🤵
執事モード

集中中のDMに執事が自動で丁寧に応対。あなたの邪魔を防ぎます。

🍝
トマト料理ランダム

返信のたびにトマト料理がランダムで登場。飽きない。

使い方

3ステップで始まる。

1
インストール

上の「Add to Slack」からワークスペースに追加。審査不要・無料。

2
コマンド実行

Slackで /pp とタイプするだけ。30分がすぐ始まる。

3
あとは集中するだけ

ステータスは変わり、DMは執事が捌く。通知が来たら休憩タイム。

コマンド一覧

全部、/ppから。

/pp デフォルト30分スタート（25分集中＋5分休憩） BASIC
/pp 60 総時間指定。5:1で自動分割（50分集中＋10分休憩） BASIC
/pp stop タイマー停止。ステータスを元に戻す。 BASIC
/pp status 残り時間の確認。 BASIC
/pp set ○○ 集中中のステータス文言をカスタム設定。 CUSTOM
